``` # Hogwarts Sorting Hat Quiz

## Setup Instructions

1. Make sure you have Python 3 installed.
2. Install Flask:
   powershell
   pip install flask
   
3. Run the backend server:
   powershell
   python main.py

4. Open your browser and go to your localhost:
   http://127.0.0.1:5000/
5. Take the quiz and see your house!
```
